Introduction

This project is an app for NCBA&E FYP in Flutter currently optimized and available for Android only. The project features a backend from Firebase.

Features

  • Latest Material UI support.
  • In-App admin panel.
  • Serverless backend (Firebase).
  • Categorial filtering.
  • Save as favorite.
  • Three-step easy news editor.
  • Full controls for admins.
  • Easy news management for publishers.
  • Discussion/Comments for each article.
  • Profile management and data control.
